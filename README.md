# 🎸 Rock Mountain – Landing Page

A simple **music festival landing page** built with **HTML and CSS**.

This project was created to practice **modern website layouts using Flexbox**, clean HTML structure, and basic styling.

---

## 📸 Preview

The page includes:

- Hero / Above-the-Fold section
- Navigation bar with social media icons
- Category menu (News / Album / Store)
- Festival ticket section
- Playlist promotion section
- Footer with newsletter subscription

---

## 🛠 Technologies Used

- **HTML5**
- **CSS3**
- **Flexbox Layout**
- **Google Fonts**

Fonts used:

- PT Sans
- Viga
